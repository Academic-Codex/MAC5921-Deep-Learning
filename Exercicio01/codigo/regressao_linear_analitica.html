Regresso Linear

Introduo

A regresso linear um dos modelos mais fundamentais em aprendizado de mquina supervisionado.
Seu objetivo estimar uma varivel de sada contnua (alvo $y \in \mathbb{R}$) a partir de variveis de entrada $x \in \mathbb{R}^d$.

Formalmente, desejamos encontrar uma funo linear da forma:

$$ h(x) = w_0 + w_1 x_1 + w_2 x_2 + \cdots + w_d x_d = w^T \tilde{x} $$

onde:

  • $\tilde{x} = (1, x_1, \dots, x_d)$ inclui o termo de bias,
  • $w = (w_0, w_1, \dots, w_d)$ o vetor de parmetros a serem aprendidos.

Funo de perda

Para avaliar a qualidade da hiptese $h(x)$, utilizamos a funo de perda quadrtica:

$$ L_D(h) = \frac{1}{N} \sum_{n=1}^N \big(h(x^{(n)}) - y^{(n)} \big)^2 $$

onde:

  • $(x^{(n)}, y^{(n)})$ so os exemplos do conjunto de treinamento,
  • $N$ o nmero total de exemplos.

A tarefa de aprendizado consiste em encontrar $w$ que minimize $L_D(h)$.

Solues

Existem diferentes estratgias para determinar o vetor de parmetros $w$:

  1. Soluo Analtica (Equaes Normais)
    Quando $d$ e $N$ no so muito grandes, podemos resolver o sistema linear: $$ w = (X^T X)^{-1} X^T y $$ onde $X$ a matriz de dados de dimenso $N \times (d+1)$.

  2. Soluo Iterativa (Gradiente Descendente)
    Para conjuntos grandes, o clculo da inversa pode ser invivel.
    O gradiente descendente ajusta $w$ iterativamente: $$ w^{(r+1)} = w^{(r)} - \eta \nabla L(w^{(r)}) $$ com taxa de aprendizado $\eta$ e gradiente: $$ \frac{\partial L}{\partial w_j} = \sum_{n=1}^N (h(x^{(n)}) - y^{(n)}) x^{(n)}_j $$

  3. SGD (Stochastic Gradient Descent)
    Em vez de usar todos os exemplos a cada iterao, atualizamos os pesos a partir de um nico exemplo (ou mini-lote).
    Isso reduz custo computacional e introduz rudo que ajuda a escapar de mnimos locais.

Avaliao da Regresso Linear

Uma mtrica amplamente utilizada o coeficiente de determinao ($R^2$):

$$ R^2 = 1 - \frac{SS_{res}}{SS_{tot}} $$

onde:

  • $SS_{res} = \sum_{n=1}^N (y^{(n)} - \hat{y}^{(n)})^2$ (soma dos resduos),
  • $SS_{tot} = \sum_{n=1}^N (y^{(n)} - \bar{y})^2$ (soma total das varincias).

Valores prximos de 1 indicam bom ajuste; valores prximos de 0 ou negativos indicam ajuste ruim.

import pandas as pd
import numpy as np
import time, psutil, os
import itertools
import torch

rows = []
cols = df_encoded.columns.tolist()

for x_col, y_col in itertools.permutations(cols, 2):  # assimtrico: xy e yx
    start_time = time.time()
    process = psutil.Process(os.getpid())

    # Converte para tensores
    x = torch.tensor(df_encoded[x_col].values, dtype=torch.float32)
    y = torch.tensor(df_encoded[y_col].values, dtype=torch.float32)

    # Monta matriz de projeto: [1, x]
    X = torch.stack([torch.ones_like(x), x], dim=1)

    # Regresso linear fechada: w = (X y)
    w = (torch.linalg.pinv(X) @ y).view(-1)  # [w0, w1]

    # Predio
    yhat = X @ w

    # Mtricas
    # ===== Estatsticas clssicas =====
    N = len(y)
    ss_res = torch.sum((y - yhat)**2)          # SSE
    ss_tot = torch.sum((y - torch.mean(y))**2) # SST
    ss_reg = torch.sum((yhat - torch.mean(y))**2) # SSR
    r2  = 1.0 - (ss_res / (ss_tot + 1e-12))
    mse = torch.mean((y - yhat)**2)

    # ===== Tempo e memria =====
    wall_time_s = time.time() - start_time
    peak_mem_mb = process.memory_info().rss / 1024**2

    # ===== Resultados =====
    rows.append({
        "kind": "pytorch",
        "x": x_col, "y": y_col,
        "w0": float(w[0]), "w1": float(w[1]),
        "R2": float(r2), "MSE": float(mse),
        "SSE": float(ss_res), "SSR": float(ss_reg), "SST": float(ss_tot),
        "N": N,
        "wall_s": wall_time_s,
        "peak_mb": peak_mem_mb,
        "iters": None,
        "grad_norm": None,
        "gap_w": None,
        "cond(X)": float(torch.linalg.cond(X))
    })

# Resultado final
result = pd.DataFrame(rows).sort_values(by=["R2", "MSE"], ascending=[False, True]).reset_index(drop=True)

if hasattr(args, "top_k") and args.top_k is not None:
    result = result.head(args.top_k)

    # tensores

    y = torch.tensor(df_num[y_col].values, dtype=torch.float32)
    X = design_matrix_1d(x)
    w = closed_form_line(X, y)

    # mtricas
    m = metrics(y, yhat)

    # para linhas suaves, ordenar por x
    order = torch.argsort(x)
    x_ord = x[order].numpy()
    yhat_ord = yhat[order].numpy()

    # mdia de Y
    y_bar = float(torch.mean(y))

    # --- PLOT ---
    plt.figure(figsize=(7.2, 5.0))

    # 1) Disperso dos dados
    plt.scatter(x.numpy(), y.numpy(), label="Dados (Y vs X)", alpha=0.8)

    # 2) Reta de regresso ( = w0 + w1*x) (traada sobre x ordenado)
    plt.plot(x_ord, yhat_ord, color="red", linewidth=2.0, label="Reta de regresso ()")

    # 3) Linha horizontal: mdia de Y
    plt.axhline(y=y_bar, linestyle="--", color="green", linewidth=1.5, label="Mdia de Y ()")

    # 4) Resduos
    xx = x.numpy()
    yy = y.numpy()
    yyhat = yhat.numpy()
    for xi, yi, ypi in zip(xx, yy, yyhat):
        # segmento vertical (do previsto  ao observado y)
        plt.vlines(x=xi, ymin=min(yi, ypi), ymax=max(yi, ypi), linewidth=0.8, color="gray", alpha=0.5)

    # 5) Ttulo e legendas
    ttl = f"{y_col} ~ {x_col}"
    if title_suffix:
        ttl += f"  {title_suffix}"
    plt.title(ttl)
    plt.xlabel(x_col)
    plt.ylabel(y_col)
    plt.legend(loc="best")
    plt.grid(alpha=0.25)

    # 6) Anotaes das mtricas no canto
